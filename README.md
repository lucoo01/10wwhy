# 十万个为什么 · 问题收集

联网收集的各领域"为什么"问题,按主题分类存放于 `docs/` 下。

## 目录结构

```
docs/
├── 医学/        # 内科/外科/骨科、妇产/儿科/男科、皮肤/五官、肿瘤/心理/中医/急救
├── 心理学/      # 心理学"为什么"问题
└── 经济学/      # 经济学"为什么"问题及原书目录
```

## 收录标准

以"为什么 / 为何 / 怎么会 / 怎么回事 / 是什么原因 / 是怎么回事"等探究原因的疑问句为主,均为真实的探究性问题(非单纯名词解释)。

## 说明

本仓库仅用于问题标题的收集与整理,内容来源于公开网络资料。如涉及版权问题,请联系删除。
